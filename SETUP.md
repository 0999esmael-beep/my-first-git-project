# HUSUMS Voting Website Setup

## Developer: Esmael Amano

### Setup Instructions:

1. **Save Profile Image:**
   - Save your profile picture as `images/profile.jpg` in the portfolio folder
   - The image should be square (recommended: 500x500px or larger)
   - Supported formats: .jpg, .jpeg, .png

2. **Open the Website:**
   - Simply open `index.html` in your web browser
   - Or use a local server for better performance

3. **Customize Content:**
   - Edit `index.html` to update text content
   - Modify `styles.css` to change colors and styling
   - Update `script.js` for interactive features
