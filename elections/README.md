# HUSUMS Elections Module

## Overview
Complete election management system for Haramaya University Students' Union (HUSUMS).

## Pages Included

### 1. **index.html** - Elections Home
- Election information and statistics
- Available positions
- How to vote guide
- Election rules and guidelines

### 2. **candidates.html** - Candidates Page
- View all candidates
- Filter by position
- Read candidate manifestos
- Candidate profiles with photos

### 3. **vote.html** - Voting Page
- Student verification form
- Secure login system
- Vote submission interface

### 4. **results.html** - Results Page
- Live election results
- Vote counts and percentages
- Visual progress bars
- Real-time statistics

## Features

✅ **Responsive Design** - Works on all devices
✅ **Position Filtering** - Filter candidates by position
✅ **Student Verification** - Secure ID verification
✅ **Live Results** - Real-time vote counting
✅ **Manifesto Viewing** - Read candidate platforms
✅ **Smooth Navigation** - Easy to use interface

## Positions Available

1. President
2. Vice President
3. Secretary General
4. Treasurer
5. Public Relations Officer
6. Academic Affairs Officer
7. Welfare Officer
8. Sports & Culture Officer

## How to Use

1. Navigate to `elections/index.html` to view election information
2. Click "View Candidates" to see all candidates
3. Click "Vote Now" to cast your vote
4. Verify your identity with Student ID and Name
5. Select your preferred candidates
6. Submit your ballot

## Customization

- Update candidate information in `candidates.html`
- Modify election dates in `index.html`
- Adjust colors in `elections.css`
- Add more positions as needed
